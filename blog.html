What is the Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface that represents the structure of a web page as a tree of objects, allowing JavaScript to access and manipulate HTML and XML documents. It enables dynamic updates to content, structure, and styling, reflecting changes in real-time on the web page. JavaScript can interact with the DOM to modify elements, attributes, and handle user interactions.

How do you select an element from DOM

You can select a DOM element using document.querySelector(), document.querySelectorAll(), getElementById(), or getElementsByClassName() based on selectors like ID, class, or tag.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM that allows you to handle events at a higher level in the DOM hierarchy, rather than attaching event listeners to individual child elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using methods like setAttribute("attribute", "value") to set or update an attribute, and getAttribute("attribute") to retrieve its value. To modify styles, you can use the style property, such as element.style.color = "red" or element.style.backgroundColor = "blue". Additionally, classList.add() and classList.remove() allow manipulation of an element's classes.
